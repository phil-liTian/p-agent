# 1. 进入代码文件夹
cd your-server  # 替换为你的代码目录名

# 2. 安装项目依赖
npm install

# 3. 启动服务（以Node.js的app.js为例）
node app.js


步骤 5：开放端口（让外网能访问）
服务启动后，需在阿里云控制台开放端口：
回到 ECS 实例页面→点击「网络与安全组」→「安全组配置」；
点击「添加安全组规则」，配置：
端口范围：填服务的端口（如 Node.js 常用的 3000）；
授权对象：填0.0.0.0/0（允许所有 IP 访问）；
保存后，浏览器输入http://47.97.58.228:3000（公网 IP + 端口），即可访问服务。
